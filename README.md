## Setting
### 1. 配置
![](https://cdn.jsdelivr.net/gh/fushaolei/imguploadtest/20200704182547.png)

你需要配置以下内容

- 用户名
- 密码
- 本地的仓库
- 远程的仓库

这么做后Imagic会再本地生成一个json文件用于存储你配置的内容
